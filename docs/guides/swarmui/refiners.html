In this guide, we will discuss the refiner feature in SwarmUI, also known as image upscaling or "high-res fix." Refiners enable users to enhance image resolution without compromising on detail. By utilizing advanced algorithms and machine learning models, refiners can significantly improve image quality.

We recommend that users familiarize themselves with the fundamentals of SwarmUI by reading our Getting Started guide prior to diving into this topic.

Refiner Parameters and Settings

The refiner feature is straightforward to use. Access the Refiner settings on the left-hand side of the interface.

The following parameters are crucial in optimizing image quality with refiners:

  • Refiner Model: This setting dictates which machine learning model to employ for image upscaling. We recommend using the default setting "(Use Base)" unless a specific reason warrants a different choice.
  • Refiner Control Percentage: This parameter controls the extent to which the refiner influences the original image. A higher value indicates greater control, while a lower value suggests less intervention. Experimentation may be necessary to achieve optimal results.
  • Refiner Method: The method of application is critical in determining the final image quality. We recommend using the default setting "PostApply," which applies the refiner after the original image has been generated. Other settings apply the refiner during the generation process, but may yield different results.
  • Refiner Upscale: This parameter determines the degree of upscaling desired for the image. Common values range from 1.5x to 4x, although higher values can be used with caution due to potential resource constraints.
  • Refiner Upscale Method: The upscale method employed by the refiner has a significant impact on image quality. We recommend using established methods such as "4x-UltraSharp" for optimal results.
