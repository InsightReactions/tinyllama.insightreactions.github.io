AI Home Server Installation

Please note that this section is not yet complete. We are actively working on it and will update it as soon as possible. Thank you for your patience!

Unpacking and Inspection

  1. Remove all components from the packaging. The packaging should contain the following (Figure 1):
    • 1x Tiny Llama AI Home Server
    • 1x Power Cable
    • 1x Network Cable
    • 1x Wi-Fi Dongle (optional)

  1. Visually inspect the components for damage. Verify there are no cracks or major structural defects on the Tiny Llama AI Home Server. The Power Cable and Network Cable should not have any exposed wiring.

Hardware Installation

The Tiny Llama AI Home Server should be placed in a well-ventilated location, near a power outlet and network port.

  1. Connect the power cable to the Tiny Llama AI Home Server (Figure 2).

  1. Connect the network cable to the Tiny Llama AI Home Server (Figure 3).

  1. Connect the corresponding ends of both cables to a power outlet and network port.

Powering On and Connecting

  1. Press the power button on the front of the Tiny Llama AI Home Server. Internal lights should illuminate, indicating power to the server. Please wait a few moments for the server to boot.
  2. Open a web browser of your choice on a device (desktop computer, laptop, or phone) and type tinyllama.local in the URL bar and press Enter to access the server's landing page. Feel free to bookmark this address; it is the primary method of accessing the tools offered by Tiny Llama.
The tinyllama.local address will only work when connected to the same network as your Tiny Llama AI Home Server. If you are unable to access this page, please ensure that both your device and the Tiny Llama AI Home Server are connected to the same local area network (LAN), and then try again.
Please note that accessing the tinyllama.local address is only possible from devices that support mDNS (Multicast DNS) resolution, which is not supported by all operating systems or network configurations. If you are unable to access the server using this method, you may connect using the IP address of your Tiny Llama AI Home Server, which can be found in the user interface of your network router's administration panel. Please refer to your router's documentation for specific instructions on how to locate this information.

Once you have the IP address, enter it into your web browser's URL bar and press Enter to access the server's landing page.

Exploring the Tiny Llama Landing Page

Upon your first visit to tinyllama.local landing page, you'll be greeted by an "Announcements and Updates" window. This window provides important information about new features and updates. It will only appear once, and then reappear when there are updates. Click or tap "Close" to dismiss this window.

The Landing Page serves as the central hub for all the tools and services offered by Tiny Llama. The top bar includes links to various sections:

  • Guides: This link leads to a section of this website that hosts basic/advanced guides and tutorials for using Tiny Llama and associated generative AI services.
  • News: This is a section of this website that covers developments in technologies that impact Tiny Llama.
  • Discuss: Registered users can provide feedback and report issues on the Tiny Llama GitHub Discussions page.
  • About: This link leads to the InsightReactions company website, which provides information about our core values, motivations, and vision for the product.

Below these links, you'll find links that take you to generative AI services hosted by the Tiny Llama platform on your local device:

  • Open WebUI: This is an extensible, feature-rich, and user-friendly self-hosted web-based AI chat interface designed for offline operation. It supports various LLM runners, including Ollama and OpenAI-compatible APIs.
  • SwarmUI: This is a modular Stable Diffusion web-based image generation interface that emphasizes making AI power tools easily accessible, performant, and extensible.
